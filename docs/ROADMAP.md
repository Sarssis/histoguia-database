# Roadmap de carga

## Regla editorial
No se incorpora un episodio a `episodes.json` si no tiene URL de episodio de Apple Podcasts con parámetro `?i=` verificado.

## Orden de ampliación recomendado
1. Completar HistoCast.
2. Completar Roma Aeterna.
3. Completar SER Historia por fases.
4. Completar La Biblioteca Perdida por fases.
5. Añadir Memorias de un Tambor si se verifican enlaces Apple.
6. Añadir otros podcasts españoles relevantes.

## Prioridad de contenido
1. Episodios que cubren huecos de La Gran Ruta.
2. Episodios monográficos de personajes/batallas/civilizaciones.
3. Episodios transversales.
4. Entrevistas o especiales.

## Criterios de calidad
- Idioma: español.
- Enlace: Apple Podcasts episodio concreto.
- Fase histórica definida.
- Tema principal normalizado.
- Al menos una etiqueta de periodo.
